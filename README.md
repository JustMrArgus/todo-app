## Launch (Deploy)

1. Launch deployed API: [LINK](https://todo-app-backend-ixp7.onrender.com/)
2. Launch deployed client: [LINK](https://todo-app-frontend-vryj.onrender.com)


## Launch (Local)

### 1. Clone the repository
```bash
git clone REPO_URL
```
### 2. Configure env file for the client
```env
NEXT_PUBLIC_API_URL=http://localhost:8000/api
```

**APP_PORT HAS TO be 8000**
### 3. Run the API
```bash
cd backend
npm run dev
```
### 4. Run the client
```bash
cd frontend
npm run dev
```
